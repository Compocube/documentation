📖 Обзор проекта

MIREA Français — веб-приложение для самостоятельного изучения французского языка, разработанное для студентов РТУ МИРЭА. Поддерживает упражнения для 3 и 4 семестров, включает интерактивные задания на порядок слов, ввод слов для разблокировки, кастомные кнопки (true/false, варианты a/b/c/d). Реализовано сохранение прогресса выполнения упражнений, адаптировано под Telegram WebView.

🎯 Основные возможности:
– Модульная система (7 модулей в семестре)
– Разделённые упражнения (части 1..N)
– Автосохранение прогресса через LocalStorage
– Поддержка ссылок в описании упражнений
– Два типа заданий: "собери порядок слов" и "введи слово для разблокировки"

Файлы упражнений подгружаются динамически с сервера:
https://mireafrancaisbot.ru/api/exercise/{Semester}/{module}.{exercise}[часть].txt
Файлы ответов имеют суффикс A (например 1.5A.txt), содержат правильную последовательность через запятую.

⚙️ Типы упражнений и распознавание

checkIfCustomExercise() — возвращает true для упражнений с предопределёнными кнопками (например, 'v'/'f', варианты a-d, технические термины). Используется жёсткий перечень упражнений:
Sem3.1.5, Sem3.3.5, Sem4.1.3 и т.д.

checkIfInputExercise() — true для упражнений, где требуется вводить слова для разблокировки (лексические задания). Список: Sem3.1.9, Sem3.2.8, Sem4.2.5 ...

setupInputExercise() — отображает поле ввода, счётчик разблокированных слов, динамические кнопки из разблокированных слов. Метод checkWordInput() нормализует ввод (сравнение по сортированным символам, без учёта дефисов/апострофов).

🌐 Загрузка файлов упражнений
GET https://mireafrancaisbot.ru/api/exercise/Sem3/1.5.txt → описание упражнения
GET https://mireafrancaisbot.ru/api/exercise/Sem3/1.5A.txt → правильный порядок слов (через запятую)
GET https://mireafrancaisbot.ru/api/exercise/Sem4/4.22.txt → для частей (упр 4.2 часть 2)

Файлы ответов опциональны. Если отсутствуют, упражнение считается без проверки (только чтение). В приложении также поддерживаются ссылки на учебники через Google Drive.

💽 LocalStorage: детальный формат
// Ключ строится: `${currentSemester}_${currentModule}_${currentExercise}_part${currentPart}`
// Пример записи:
{ "Sem4_5_3": { selectedWords: ["v", "f", "v"], correctOrder: ["v", "v", "f"], unlockedWords: [], allWords: [], isCustomExercise: true, isInputExercise: false, lastUpdated: "2025-04-24T10:30:00Z" } }
Прогресс очищается только после успешного выполнения упражнения (вызов clearCurrentProgress).
📱 Интеграция с Telegram Web App

При инициализации проверяется наличие глобального объекта Telegram. Вызываются методы:

if (typeof Telegram !== 'undefined') { Telegram.WebApp.ready(); // сообщает, что приложение загружено Telegram.WebApp.expand(); // растягивает мини-апп на весь экран }

Это позволяет корректно работать внутри Telegram-бота, скрывая стандартные отступы.

buttons.forEach(btnText => {

});
Пример: проверка ответа (правильная последовательность)
// correctOrder из файла 1.5A.txt: "Bonjour,je,m'appelle"
// Пользователь выбрал: ["Bonjour","je","m'appelle"] → checkAnswer() → успех
Важно: При input-упражнении сравнение слов происходит через нормализацию (сортировка букв), поэтому "l'écran" и "lécran" считаются совпадающими.
